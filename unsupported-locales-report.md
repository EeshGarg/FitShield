# FitShield v0.49 Unsupported Requested Locales

Chrome extension `_locales` only supports the locale codes listed in the Chrome
i18n documentation. From the requested Americas, Africa, and Oceania expansion,
these locale codes were supported and added:

- `pt_BR` - Portuguese (Brazil)
- `sw` - Swahili
- `am` - Amharic

The following requested locale codes were not created because Chrome extension
`_locales` does not support them:

## Americas

- `qu` - Quechua
- `gn` - Guarani
- `ay` - Aymara
- `ht` - Haitian Creole
- `iu` - Inuktitut
- `chr` - Cherokee
- `haw` - Hawaiian
- `mi` - Maori
- `sm` - Samoan

## Africa

- `ha` - Hausa
- `ig` - Igbo
- `yo` - Yoruba
- `zu` - Zulu
- `xh` - Xhosa
- `st` - Sesotho
- `sn` - Shona
- `rw` - Kinyarwanda
- `so` - Somali
- `mg` - Malagasy
- `ny` - Chichewa
- `ti` - Tigrinya
- `om` - Oromo
- `ff` - Fulah
- `wo` - Wolof

## Oceania

- `fj` - Fijian
- `to` - Tongan
- `ty` - Tahitian

Source checked: https://developer.chrome.com/docs/extensions/reference/api/i18n
